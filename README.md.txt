# Rent_car_analysis

Une application pour extraire les prix d'achat depuis le site de vente de voiture lacentrale.com, les prix de location de voitures depuis le site getaround.com, ainsi que l'analyse de rentabilité

## Fonctionnalités
- Extraction de données web avec Selenium (Privé)
- Génération de fichiers xlsx pour les modèles et marques. (Privé)
- Analyse de rentabilité à la base des données extraites.
